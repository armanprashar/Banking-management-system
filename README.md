Banking Management System

The Banking Management System is a desktop-based application developed using Java Swing, JDBC, and MySQL to simulate real-world banking operations with a secure and user-friendly interface. The project follows a layered MVC-inspired architecture that separates the user interface, business logic, and database operations for better maintainability and scalability. It provides secure authentication with password hashing, role-based access for administrators and customers, and efficient database connectivity using JDBC and PreparedStatement.

The application allows users to perform essential banking activities such as account creation, money deposit, withdrawal, fund transfer, balance inquiry, and transaction history management. It also includes an admin dashboard where administrators can monitor user activities, manage customer accounts, freeze or unfreeze accounts, and track banking transactions through a centralized ledger system. JDBC transaction management is implemented to ensure data consistency and reliability during financial operations.

The graphical user interface is built using Java Swing and includes modern usability features such as validation dialogs, responsive loading using SwingWorker, searchable transaction history, and dark/light theme support. The project demonstrates practical implementation of Object-Oriented Programming concepts, database management, secure authentication, exception handling, and desktop application development, making it a strong portfolio project for Java developer and software engineering roles.
